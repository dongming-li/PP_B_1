    -- Build System Documentation --

    Overview

    This is a basic prototype of the builder used for the 309 project. The builder is a Zeus style builder which patches the binary at certain offsets. The builder produces an agent.exe file from the package.zip file which contains both a manifest as well as the template binary for the builder to use.

    Builder Usage

    You must have python 3 installed as well as the pefile python library in order to run the builder application. It is a command line application which will display prompts on the configurable variables. In the future we plan to have the web app generate the user interface for configuration based on the manifest file bundled with the package.

    Installer Usage

    After running the builder.py the builder will output an agent.exe file. Copy this agent.exe into the installer folder and zip it up. Run the install.bat script on the target system
